Site mobile iPhone

Ce fichier comporte :
- un fichier HTML5 index.html
- un dossier css avec :
    - un fichier CSS3 style.css
    - un sous-dossier images comportant deux images
- un dossier js avec un fichier JS script.js

C'est une reproduction d'un mobile iPhone de la marque Apple.
Il a été créé uniquement en HTML5 et CSS3.

On peut allumer et éteindre le téléphone à l'aide du bouton situé en bas au milieu du mobile.
Une fois allumé, on voit apparaître : 
- une barre de recherche Google (où il est possible de taper du texte à l'intérieur), 
- une météo,
- une heure et une date,
- une barre de notifications,
- des applications mobiles apparaissant l'une après l'autre.
